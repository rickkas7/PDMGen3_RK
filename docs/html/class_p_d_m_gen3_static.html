Detailed Description

template<size_t NUM_SAMPLES>
class PDMGen3Static< NUM_SAMPLES >

Construct a PDM decoder using a static buffer.

Parameters
NUM_SAMPLESThe number of samples to save. The number of bytes is 4x this value because each sample is 2 bytes and there are double buffers. The number is must be large enough so you can process the data from loop() without losing samples.
